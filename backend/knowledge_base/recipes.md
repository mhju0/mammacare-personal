# 맘마케어 레시피 목록

맘마케어 앱에 등록된 이유식 레시피 정보입니다.

## 레시피: 고구마 당근죽

- 이유식 단계: 초기 이유식 (5~6개월)
- 설명: 고구마, 당근을 잘게 다진다
불린 쌀과 함께 물을 붓고 중약불에서 끓인다
재료가 푹 익고 농도가 잡힐 때까지 저어가며 죽을 끓인다
- 재료: 고구마, 당근

## 레시피: 고구마 애호박죽

- 이유식 단계: 초기 이유식 (5~6개월)
- 설명: 고구마, 애호박, 쌀을 잘게 다진다

## 레시피: 고구마미음

- 이유식 단계: 초기 이유식 (5~6개월)
- 설명: 불린 쌀과 고구마는 곱게 갈아둔다

완전히 끓으면 농도를 확인한 후 고운체에 거른다
- 재료: 고구마 10g, 쌀 10g

## 레시피: 기장미음

- 이유식 단계: 초기 이유식 (5~6개월)
- 설명: 쌀과 기장은 찬물에 불린 뒤 물기를 빼고 분말기에 담아 곱게 간다
냄비에 곱게 간 쌀과 기장, 물을 넣고 센 불에서 저어가며 끓인다
끓기 시작하면 불을 약하게 줄여 쌀과 기장이 부드럽게 퍼질 때까지 끓인 뒤 불을 끄고 고운체에 거른다
- 재료: 기장 5g, 쌀 10g

## 레시피: 단호박 양배추미음
